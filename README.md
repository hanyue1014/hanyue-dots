# hanyue-linux-config
This repo contains most of my linux config files
---
## .zshrc
`.zshrc` is the config file for my zsh
#### Prerequisites
- zsh or zshell
- oh-my-zsh plugin
  - Plugins installed via oh-my-zsh
  - Spaceship Prompt (theme for zsh)
  - zsh-syntax-highlighting
  - zsh-autosuggestion
